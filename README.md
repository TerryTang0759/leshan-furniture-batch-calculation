# 🏠 乐山家具定制 - 柜体投影面积报价计算系统（集成版）

## 📋 项目简介

这是一个集成了柜体拆板计算和投影面积单价计算的完整工具。系统可以自动计算柜体拆板总面积和总价，然后计算投影面积单价，精确到元。无需手动输入拆板总价，一体化完成所有计算。

## ✨ 核心功能

### 🎯 主要用途
- **一体化计算**: 自动计算柜体拆板总价和投影面积单价
- **快速报价**: 输入尺寸和材质即可获得精确报价
- **价格对比**: 不同柜体尺寸和材质的单价对比
- **精确计算**: 结果精确到元，便于报价

### 📐 计算公式
```
拆板总面积 = 顶板+底板+左右侧板+背板+横隔板+竖隔板
拆板总价 = 拆板总面积 × 材质单价
投影面积单价 = 拆板总价 ÷ (柜体宽度 × 柜体高度)
投影面积 = 柜体正面投影面积（宽 × 高）
```

## 🚀 使用方法

### 系统一：单个柜体计算
直接在浏览器中打开 `index.html` 文件

### 系统二：批量投影面积计算（推荐）
运行批量计算系统：
```bash
./启动批量计算系统.sh
```

或者手动启动：
```bash
npm install
node api.js
```

然后访问：http://localhost:3001

### 批量计算系统使用步骤

#### 1. 输入标准尺寸
- **柜体高度**: 输入高度（单位：mm）
- **柜体宽度**: 输入宽度（单位：mm）
- **柜体深度**: 输入深度（单位：mm）

#### 2. 自动计算
系统自动计算：
- **横隔板数量** = ⌊高度/400⌋ + 1（含顶底板）
- **纵隔板数量** = ⌊宽度/1000⌋ + 1（含左右侧板）

#### 3. 批量计算
点击"开始批量计算"，系统将：
- 从MongoDB加载所有柜体材质数据
- 计算每种材质的拆板总价
- 计算投影面积单价
- 按价格排序显示结果

#### 4. 查看结果
- **统计信息**: 材质总数、平均单价、最低/最高单价
- **对比表格**: 所有材质的投影面积单价对比
- **排序显示**: 按投影面积单价从低到高排序

### 4. 操作按钮
- **🧮 计算投影面积单价**: 执行计算
- **🗑️ 清空输入**: 清空所有输入框

## 📊 计算示例

### 示例1：标准衣柜
- 柜体宽度: 180cm
- 柜体高度: 240cm
- 拆板总价: 3500元
- **结果**: 投影面积单价 = 3500 ÷ (1.8 × 2.4) = 3500 ÷ 4.32 = 810元/㎡

### 示例2：书柜
- 柜体宽度: 120cm
- 柜体高度: 200cm
- 拆板总价: 2800元
- **结果**: 投影面积单价 = 2800 ÷ (1.2 × 2.0) = 2800 ÷ 2.4 = 1167元/㎡
